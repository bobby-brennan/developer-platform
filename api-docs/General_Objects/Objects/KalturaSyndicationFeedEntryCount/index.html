DocumentationKalturaSyndicationFeedEntryCount

SchemaExample
{
  "actualEntryCount": 0,
  "requireTranscodingCount": 0,
  "totalEntryCount": 0
}
object:KalturaSyndicationFeedEntryCount
actualEntryCount:integer

count of entries that will appear in the feed (including all relevant filters)

requireTranscodingCount:integer

count of entries that requires transcoding in order to be included in feed

totalEntryCount:integer

the total count of entries that should appear in the feed without flavor filtering
